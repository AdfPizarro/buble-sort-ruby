# BUBBLE S0RT
​
## Built With
​
- Ruby 2.6.5
​
​
## Getting Started
​
To get a local copy up and running follow these simple example steps.
​
git clone https://github.com/AdfPizarro/buble-sort-ruby

​
### Usage
​
type 
 ruby bubble.rb

​
## Commands
- rubocop bubble.rb
​
Note: You need to do `bundle install` before executing the commands above.
